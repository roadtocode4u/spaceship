# Relational Operator & Logical Operators

## Relational Operator : (>,<,>=,<=,==)

- comparision
- true / false value

```cpp
  1. == is equals to
ex : a == b // == comparision

a = b // value assign

two values are equal then result is true

two values are differebt then result is false
```

**💻 Example 1 :**

```cpp

using namespace std;
```

## Logical Operator : (&&,||, !)

```cpp
 1. Logical AND  => &&
 2. Logical OR =>  ||
 3. Logical NOT(Negation) => !
```

```cpp
1. Logical AND (&&)

**Truth Table**
(condition 1 && condition 2)
 T   T => T
 T   F => F
 F   T => F
 F   F => F
```

**💻 Example 1 :**

```cpp
#include <iostream>
using namespace std;

int main() {

   int a = 5;

   int b = 8;

   int res =  a==5 && b==a;

   cout<<res;

   return 0;
}
```

**⚙️ Output :**

> 1

**💻 Example 2 :**

```cpp

using namespace std;

int main() {

   int a = 5;

}
```

> 0

```cpp
2. Logical OR (||)

**Truth Table**
(condition 1 || condition 2)
 T   T => T
 T   F => T
 F   T => T
 F   F => F
```

**💻 Example 1 :**

```cpp
#include <iostream>
using namespace std;

   int jee = 130;

   int cet = 20;

   int res = jee >= 120 || cet >= 200;

   cout<<res;

   return 0;
}
```

**⚙️ Output :**

> 1
