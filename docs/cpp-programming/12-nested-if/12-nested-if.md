## Homeworks Examples :

**💻 Example 1️⃣ : WAP to calculate largrst number among 3 using if-else-ladder ?**

```cpp
#include <iostream>
using namespace std;

int main() {

    int a,b,c;
    cout<<"Enter 3 Numbers:";
    cin>>a>>b>>c;

    if(a>b && a>c)
    {
        cout<<"A is largest";
    }
    else if(b>a && b>c)
    {
        cout<<"B is largest";
    }
    else if(c>a && c>b)
    {
        cout<<"C is largest";
    }
    else
    {
        cout<<"All are equal";
    }
     return 0;
}
```

**⚙️ Output :**

> Enter 3 Numbers 5 80 7<br/>
> B is largest
